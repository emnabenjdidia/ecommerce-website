#  TechShop — E-Commerce Website

A full-stack e-commerce platform for selling electronics products
built during my internship.

##  Technologies Used
- **Front-end:** HTML, CSS, JavaScript, Bootstrap
- **Back-end:** Node.js
- **Database:** SQL
- **API Testing:** Insomnia

##  Pages
-  Home
-  Hot Deals
-  Products
-  About Us
-  Contact

##  Features

###  Customer Side
- Browse products by category 
- Detailed product pages with descriptions and specs
- Shopping cart and order placement
- User account management

###  Admin Dashboard
- Overview: Total Users, Total Orders, Total Revenue
- Mark orders as Paid / Cancel / Delete
- View all users and their orders
- Add, edit, and delete products
- Edit admin profile / delete account
